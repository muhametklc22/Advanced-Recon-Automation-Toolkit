# Advanced-Recon-Automation-Toolkit
Bu araç, pentest ve bug bounty aşamasında bilgi toplama (reconnaissance) sürecini hızlandırmak için geliştirilmiş otomatik bir keşif panelidir.  Subdomain analizi, dizin taraması, CMS tespiti, WAF analizi, geçmiş URL toplama, JS endpoint analizi ve port–zafiyet eşleştirme gibi işlemleri tek yerden yönetmeyi sağlar.  

 git clone https://github.com/muhametklc22/Advanced-Recon-Automation-Toolkit
cd Advanced-Recon-Automation-Toolkit
chmod +x recon.py
python3 recon.py
